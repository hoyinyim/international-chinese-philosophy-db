# 國際中國哲學英文期刊資料庫  
International Chinese Philosophy Journal Database

本專案是以「國際中國哲學英文期刊閱讀」為核心的長期學術訓練與資料庫建置計畫。其目標不是建立一般讀書筆記，而是將每日閱讀的英文中國哲學、國際漢學、比較哲學與東亞思想研究論文，轉化為可累積、可分類、可檢索、可供 AI 工具調用的學術資料。

## 一、核心資料庫

本專案包含兩套核心資料庫：

### 1. 國際中國哲學學術語用資料庫

此資料庫用於保存英文學術論文中值得學習的字詞、語塊、句式與論述策略。重點不在於蒐集「漂亮英文」，而在於分析作者如何使用英文進行立場界定、問題提出、文獻定位、概念界定、論證推進、轉折與修正、反駁與批判、方法說明、材料處理、比較與對照、限制與保留、結論與貢獻。

### 2. 國際中國哲學研究視野資料庫

此資料庫用於保存每篇論文的研究問題、研究對象、文本材料、方法進路、學術對話對象、國際學術視野、研究啟發與可延伸問題。

## 二、未來 AI 工具目標

本專案未來將發展兩個 AI 對話入口：

1. 國際中國哲學語用  
   使用者輸入想表達的中文學術意思後，AI 從語用資料庫中搜尋相關英文詞彙、語塊、句式與論述策略，並說明其語氣、強度、適用情境與限制。

2. 國際中國哲學視野勘探  
   使用者輸入研究主題後，AI 從研究視野資料庫中搜尋相關概念、文本、方法與學術議題，回報國際中國哲學學界的研究方向、材料使用、方法特徵與可開發問題。
